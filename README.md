# Aplikasi Absensi Organisasi Kampus

## Deskripsi Aplikasi

Aplikasi Absensi Organisasi Kampus adalah aplikasi mobile berbasis Flutter yang digunakan untuk mencatat dan mengelola data kehadiran anggota organisasi kampus. Aplikasi ini memungkinkan pengguna untuk menambahkan, melihat, mengedit, dan menghapus data absensi.

---

## ✨ Fitur Aplikasi

Fitur utama dalam aplikasi ini:

### 1. Tambah Data Absensi (Create)

Pengguna dapat menambahkan data absensi baru dengan mengisi form yang tersedia, yaitu:

* Nama Anggota
* NIM
* Divisi
* Status Kehadiran

### 2. Lihat Data Absensi (Read)

Pengguna dapat melihat seluruh daftar absensi yang telah ditambahkan dalam bentuk list.

### 3. Edit Data Absensi (Update)

Pengguna dapat mengedit data absensi yang sudah ada.

### 4. Hapus Data Absensi (Delete)

Pengguna dapat menghapus data absensi dari daftar.

### 5. Multi Page Navigation

Aplikasi menggunakan beberapa halaman:

* Halaman Home (Daftar Absensi)
* Halaman Tambah Absensi
* Halaman Edit Absensi
